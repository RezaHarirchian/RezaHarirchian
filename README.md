### 🛡️ [ServerSpy](https://serverspy.ca)
**AI-Powered Server Monitoring Platform**

> *Your infrastructure, watched 24/7 by AI — so you don't have to be.*

ServerSpy automatically scans your servers every 24 hours for **security vulnerabilities**, **cloud cost waste**, and **behavioral anomalies**. The AI engine behind it — **SpyCore AI** — turns raw server data into clear, actionable insights without needing a DevOps team.

Built entirely solo: architecture → development → production deployment.
